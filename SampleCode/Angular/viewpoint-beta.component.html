Procedures
This section is intended to outline the procedures used for what you see on this screen. This will help guide the user on how this page is intended to be used. This is particularly useful for things like the Revenue report where we want to spell out the logic used to calculate each revenue stream, and provide a quick reference to where we can find documentation and approvals when needed (in case of SOX controls and logs).
Example procedure:
Revenue eligibility in FY 2020 starts with a “completed” stage where the contractor has certified that the installation has been completed. The revenue streams collected for this program and their calculations are as follows:
Incentive:
Measure level incentives are aggregated at the work order level per the approved program incentive list effective at the date of audit/sale
Customer Copay:
0.33 times the total incentive for that work order
Performance Fee:
Measure level savings are aggregated at the work order level and multiplied by the agreed performance rate of $0.20/kWh per client contract.
Adjustments:
As work orders continue to be updated until after post-inspection, each new revenue report will reconcile these changes and update accordingly.

ViewPoint-Assessment Tool
Our ViewPoint Assessment tool is mobile friendly and has four main modules: Lead, Audit, Construction, Inspection. This tool focuses on field activities and uses the same customer and measure data within each module, so if one is updated it is reflected throughout. The key benefits of this tool include both offline and online capabilities, with API calls for customer eligibility and measure offerings. This syncs with ViewPoint’s main hub for additional tracking capabilities.
Features you can see today
Lead Mode
Track customer lead visits from initial interest in participating to collecting basic customer information and documenting notes/interactions with the customer.
The lead mode of the assessment tool uses a site survey to assess if potential customers qualify for the program.
Main users of this mode will be Community Based Organizations (CBO) or Willdan’s Energy Service Representatives (ESR). Interested leads will be assigned to an ESR/Auditor who will schedule an assessment.
Calendar feature: Organize, plan, and track Events and Tasks. Never miss an appointment or deadline with Viewpoint Calendar.

ViewPoint-Willdan
ViewPoint-Willdan is our program office module intended for Willdan office staff that gives full visibility into all aspects of the program. In ViewPoint-Willdan you can setup organizations and contractor profiles, create user roles, assign customers or tasks to users, see all the project data, and much more.
Features you can see today
Customer search: query customers, prospects or projects through various filtering options. View business locations using the Interactive Mapping Feature and plot an efficient schedule for daily field activities.
Setup and manage external organizations / contractors and track licenses, insurance, trade/expertise, capacity, and other onboarding requirements. Includes ability to create and track access configuration for union auditors, installers, technicians, etc.

The ViewPoint Table
With every update, we will be highlighting a new component of our Viewpoint design as well as its main features and benefits. The highlight of this first release is our ViewPoint Table. Different than a regular table, this advance table features custom filtering, column sorting, searching, pagination, column reordering and responsive extension, among others. Try it out for yourself below!
Table Features
Function Description
Column Sort Click on any column to sort content in ascending or descending order
Column Filters Apply 1 or more Column Filters per 1 or more Columns from the Result Set table. Use the additional Filters to further hide the number of records not needed on the display.
Expand/Shrink Column View Drag left column border to the left or right which will widen or shrink column width. Helpful in viewing truncated column values.
Rearrange Columns Horizontally drag columns (left or right) to move field(s) in preferred display order.
Ellipse (3 Vertical Dots) Submenu Click on the Ellipse Submenu at the end of each row to view additional Options.
Page Display Navigation Double left/right arrows move display to 1st or Last page of results. Single left/right arrow displays Next or Previous page display.

Development |

ViewPoint modules currently in development
Account Overview - an intuitive and detailed interface that displays a customer’s information with their associated audit(s), project(s), and workorder(s).
Audit UI - a module under ViewPoint’s Assessment tool for conducting energy audits by either internal or external auditors.
Contractor Pricing - a program admin ViewPoint interface to allow program management staff to setup, view and manage contractor pricing information.
Mobile Design – a scalable ViewPoint mobile user interface that adjusts automatically to the different mobile device sizes regardless of the operating system.

Requirements |

ViewPoint modules in the requirements analysis/verification phase
Construction UI - a module under ViewPoint’s Assessment tool to capture work performed by contractors such as installation scope of work, labor hours, material needs, and project completion forms (PCF).
Labor Cost – an internal Willdan ViewPoint interface for managing all labor costs associated with a workorder.
ESR Proposal – part of the Audit module under ViewPoint’s Assessment tool, it allows the user to generate proposals for customers based on the audit scope, with capabilities to add/remove line items from the scope and obtain signatures for work authorization and other sales’ required documents.

Modules on Deck |

ViewPoint modules waiting to enter requirements queue
Inspection/QV – a module under ViewPoint’s Assessment tool for conducting inspections/quality assurance on installed energy projects by QA specialists.
Work Order – an intuitive and detailed interface that displays work order details details such as the installation scope of work, energy savings, labor cost, material cost, measure quantity, and related documentation.
Safety – measure contractor performance based on key performance indicators such as safety practices, compliance to OSHA regulations, and safety training requirements.
